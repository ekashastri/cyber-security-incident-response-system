# Cyber Security Incident Response System

A console-based cyber security incident response system developed using Java and Object-Oriented Programming concepts.

## Features
- Incident reporting and tracking
- User authentication
- Analyst assignment
- Exception handling
- Modular project structure

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File Handling

## Project Structure
- model
- service
- util
- exception

## Concepts Applied
- Classes and Objects
- Inheritance
- Exception Handling
- Modular Programming
